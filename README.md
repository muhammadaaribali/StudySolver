## ✨ Features
- 📚 **Responsive Design** – Works smoothly on desktops, tablets, and mobile devices.
- 🎯 **Student-Centered UI** – Easy navigation and visually appealing layout for learners.
- ⚡ **Bootstrap Components** – Pre-built UI elements for quick interaction.
- 📱 **Media Query Optimization** – Customized styles for different screen sizes.
- 🔍 **Organized Study Content** – Sections for notes, assignments, and practice material.

---

## 🛠️ Tech Stack
- **HTML5** – Structured content and semantic elements
- **CSS3** – Styling and layout customization
- **Bootstrap** – Responsive grid system and UI components
- **Media Queries** – Device-specific styling
